Component Library Button

Inputs

  
    variant: 'primary' | 'secundary' => default 'primary'
    size: 'sm' | 'md' | 'lg' => default 'lg'

Use

  
    <lib-button>
      With input variabels primary and md
    </lib-button>

    or

    <lib-button [variant]="'primary'" [size]="'md'">
      With input variabels primary and md
    </lib-button>

Render

No input variabels | With input variabels primary and md | With input variabels secundary and sm
